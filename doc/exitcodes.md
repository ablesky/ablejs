### exit codes for ablejs

>it follows the gruntjs exit codes: [grunt exit codes](http://gruntjs.com/api/exit-codes)

#### Exit codes

* 0 - No errors! 
* 1 - Fatal error
* 2 - Missing gruntfile error
* 3 - Task error
* 4 - Template processing error
* 5 - Invalid shell auto - completion rules error
* 6 - Warning
